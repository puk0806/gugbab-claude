## 8. 편집장(Editor) 대응

### 8-1. 심사위원 의견 충돌 시

심사위원 1과 2의 권장이 정반대인 경우:
- 응답서에 양쪽 코멘트 모두 인용
- 본인의 학술적 입장을 근거와 함께 제시
- **편집장에게 판단 요청**: "Given the divergent suggestions, we have followed [선택한 입장] for [근거], and welcome the editor's guidance."

### 8-2. Reject에 대한 Appeal

| 상황 | 권장 |
|------|------|
| 심사위원 코멘트에 명백한 사실 오류 | Appeal 가능 (드물게) |
| 단순히 결과가 마음에 안 듦 | ❌ Appeal 금지 — 다른 학술지 재투고 |
| 심사가 학술지 scope 밖에서 평가됨 | Appeal 가능 |

**Appeal letter 구조:**
1. Manuscript ID, 결정 일자
2. Appeal 사유 (사실 오류 / scope 오해 등) — 감정적 어조 금지
3. 구체적 근거 (원고 페이지·줄, 인용 자료)
4. 요청: 재심사 또는 추가 심사위원 배정
5. 마감 통상 30일 이내 — 학술지 정책 확인

> Appeal 성공률은 낮다. 다른 학술지 재투고가 통계적으로 빠른 게재 경로다.

## 9. 투고 시스템별 실무 팁

### 9-1. 영문 학술지 — ScholarOne / Editorial Manager

| 시스템 | 사용 출판사 (예) |
|--------|------------------|
| **ScholarOne Manuscripts** | Taylor & Francis, Wiley, Oxford 등 다수 |
| **Editorial Manager** | Elsevier, Springer Nature 등 다수 |
| **EJP** | Wolters Kluwer 등 |

**공통 절차:**
1. 계정 생성 → ORCID 연결 (대부분 학술지에서 권장 또는 필수)
2. 원고 업로드 (manuscript / figures / tables / supplementary 분리 업로드)
3. 메타데이터 입력 (제목, 초록, 키워드, 저자 정보)
4. 추천/배제 심사위원 입력 (선택)
5. 윤리 선언 체크박스
6. Cover letter 업로드
7. PDF 빌드 → 미리보기 확인 → 최종 제출

> **ORCID는 사실상 필수**다 (publisher마다 정책 다름). 사전 등록: https://orcid.org/

### 9-2. KCI 등재지 — JAMS / 자체 시스템

- JAMS 포털: https://portal.jams.or.kr/
- 학회마다 서브도메인 (`{학회}.jams.or.kr`) 또는 자체 시스템.
- 회원가입 → 학회 회원 인증 → 논문 투고
- 학회비·심사료·게재료 사전 확인 (학회별 차이 큼).
- 심사 결과는 **편집위원회 심의 후** 통보 — 단일 심사위원 결과가 아님.

## 11. 도덕윤리교육 분야 특화 팁

### 11-1. 영문 학술지 투고 시

- **Scope 적합성 검증**: Journal of Moral Education은 도덕 발달·도덕 추론·인성교육 등 광범위하지만, 단순 교과교육론은 scope 밖일 수 있음. 최근 3년 목차 정독 필수.
- **이론적 기여 강조**: 영문 도덕교육 학술지는 경험 연구도 받지만 **이론적·철학적 기여**를 cover letter에서 명시하면 desk review 통과율 상승.
- **방법론 다양성**: 양적/질적/혼합/철학적 분석 모두 게재되므로 자신의 방법론에 맞는 학술지 선정.

### 11-2. KCI 등재지 투고 시

- **선행연구 인용**: 한국 도덕교육학계 주요 학자(예: 분야 표준 문헌)의 인용 비중을 적정 유지.
- **학위논문 사용 사전 공시**: 한국도덕윤리과교육학회·한국도덕교육학회 등은 학회별로 정책 다름 — 투고 규정 반드시 확인.
- **연합 학술대회 발표 후 투고**: 학술대회 발표 논문을 같은 학회지에 투고할 때 사전 공시 필요.
