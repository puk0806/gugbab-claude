SEO · GEO 유의사항 치트시트

2026-06-04 기준 · 공식 문서 교차 검증 완료 스킬 기반 · 한 눈에 보는 실무 핵심 요약

🤖 GEO — AI 검색 최적화 (Generative Engine Optimization) NEW 2024+

ChatGPT·Perplexity·Google AI Overviews·Claude 답변에 사이트가 인용되도록 하는 최적화. 전통 SEO와 목표가 다르다.

⚠️ GEO의 핵심: 학습용 크롤러를 전부 차단하면 AI 답변 인용 기회도 함께 사라진다. 학습은 차단하되 인용·검색용은 허용하는 전략이 기본값.

AI 크롤러 3분류 + robots.txt 전략

분류User-agent 예시차단 시 영향권장
학습용GPTBot ClaudeBot Google-Extended Applebot-Extended Meta-ExternalAgent CCBot모델 학습에 미사용 — 인용 영향 적음선택적 차단 가능
인덱싱·검색용OAI-SearchBot Claude-SearchBot PerplexityBot차단하면 AI 검색 결과에서 사라짐허용 권장
사용자 요청 fetchChatGPT-User Claude-User Perplexity-User사용자가 명시 요청 시 동작. 일부는 robots.txt 무시 가능 명시허용 권장
ℹ️ Google-Extended는 크롤러가 아니라 robots.txt 토큰이다. Googlebot 자체가 아닌, 수집 데이터가 Gemini·Vertex AI 학습에 쓰일지만 제어한다. Googlebot을 차단하면 일반 구글 검색도 사라진다.

인용 친화 콘텐츠 구조 (GEO 핵심)

✅ AI가 인용하기 좋은 구조
  • 질문-답변 구조 우선 (What is X? → 명확한 1~2문장 답변)
  • 청크 100~300토큰 단위로 단락 구분
  • FAQPage JSON-LD 마크업 추가
  • 정의·통계·인용 출처를 명시 (신뢰도 신호)
  • 문장 두괄식: 핵심 → 세부 설명 순서
  • 고유 데이터·조사·관점 포함 (복붙 콘텐츠 인용 안 됨)
❌ AI 인용이 안 되는 패턴
  • 광고성 서술 ("최고의 솔루션!" 등)
  • robots.txt로 OAI-SearchBot 차단
  • JavaScript 렌더링 의존 (크롤러가 읽지 못함)
  • 중복 콘텐츠 (타 사이트 복붙)
  • 로그인 필요 페이지
  • 이미지 속 텍스트만 (OCR 의존 불안정)
⚠️ llms.txt는 Answer.AI가 제안한 비공식 표준이다. 아직 OpenAI·Google·Anthropic이 공식 채택하지 않았으므로 "있으면 좋지만 없어도 현재 SEO에 직접 영향 없음" 수준.

OG (OpenGraph) 필수
  • og:title — 페이지 제목
  • og:description — 요약 (200자 이내)
  • og:image — 1200×630px 절대 URL
  • og:url — canonical과 동일
  • og:type — website or article
X (Twitter) 카드
  • twitter:card — summary_large_image
  • twitter:title
  • twitter:description
  • twitter:image — 최소 300×157px
  • twitter:site — @계정명
ℹ️ meta name="keywords"는 Google이 무시. 작성해도 SEO에 해롭진 않지만 시간 낭비.

⚡ Core Web Vitals (CWV)

지표측정 대상GoodNIPoor
LCP가장 큰 콘텐츠 렌더 시점≤ 2.5s2.5–4.0s> 4.0s
INP모든 상호작용 응답성 (FID 대체, 2024-03-12)≤ 200ms200–500ms> 500ms

⚠️ Lighthouse 90+이어도 CrUX(실제 사용자 데이터)가 Poor일 수 있다. Google 순위 반영은 CrUX 기준. Lighthouse는 디버깅·CI 보조용.
LCP 개선 핵심
  • 히어로 이미지에 fetchpriority="high"
  • <link rel="preload" as="image"> 조기 선언
  • LCP 이미지에 loading="lazy" 금지
  • TTFB 500ms 이하 유지 (서버 응답 최적화)
  • 렌더 블로킹 CSS·JS 제거 or defer
INP 개선 핵심
  • React: useTransition / useDeferredValue
  • scheduler.yield() (Chrome 129+ stable)
  • Long Task > 50ms를 Task로 쪼개기
  • 무거운 이벤트 핸들러는 setTimeout(0) 위임
  • Third-party 스크립트 → Partytown 격리
CLS 개선 핵심
  • 모든 이미지에 width + height 명시
  • 광고·임베드: 고정 크기 placeholder
  • 동적 삽입 콘텐츠는 뷰포트 상단 위치 금지
  • 웹폰트 FOUT: font-display: swap + size-adjust
  • 애니메이션은 transform·opacity만

  • 금지: "이미지", "사진", 파일명 반복
  • 장식용 이미지: alt="" (공백 필수)
  • 상품 이미지: 색상·각도·모델번호 포함

🤖 봇 관리 (Bot Management)

Googlebot 정당성 검증 (4단계)
  • ① 접속 IP 역DNS 조회 → hostname 확인
  • ② hostname으로 정방향 DNS 조회 → IP 일치 확인
  • ③ Googlebot IP 범위 JSON 공식 경로에서 확인
  • ④ User-agent 스푸핑 가능 → IP 검증이 실질적 방법
Cloudflare · AWS WAF 구분
  • Cloudflare: cf.verified_bot 불리언 필드
  • Cloudflare 카테고리: cf.verified_bot_category
  • AWS WAF: awswaf:managed:aws:bot-control:bot:category:search_engine
  • Naver 크롤러 UA: Yeti (공식 명칭)
ℹ️ Googlebot IP 범위 공식 JSON: static/crawling/ipranges/common-crawlers.json (검색엔진 공식 문서에서 최신 경로 확인 권장)

🔗 Canonical · 리다이렉트 · URL 구조

Canonical 3원칙
  • 자기참조 canonical을 모든 페이지에 추가
  • 절대 URL 사용 (상대 URL 금지)
  • https/www/trailing slash를 통일한 정규화 형태
리다이렉트 유형
  • 301 — 영구 이동 (link equity 전달 ~99%)
  • 302 — 임시 이동 (link equity 보류)
  • 307 — 임시, 메서드 유지 (POST 보존)
  • 체인 리다이렉트 3단계 초과 금지
  • 리다이렉트 루프 모니터링 필수
Sitemap 한계
  • 최대 50,000 URL / 50MB (압축 전)
  • 초과 시 sitemap 분할 + sitemap index 파일
  • lastmod는 실제 변경일만 (형식적 날짜 금지)
  • rel=prev/next: 2019년 Google이 무시 선언 (폐기)
내부 검색 결과 처리
  • 파라미터 URL 중복 위험 → noindex 권장
  • Faceted navigation: canonical 또는 noindex
  • 정렬·필터 파라미터: GSC URL 파라미터 설정

📱 모바일 · PWA SEO

모바일 필수 조건
  • viewport 메타 필수 (initial-scale=1)
  • 탭 타겟 최소 48×48px (SC 2.5.8)
  • 텍스트 최소 16px (확대 불필요)
  • 가로 스크롤 금지
  • Google은 모바일 우선 색인(Mobile-First Indexing)
PWA + SEO 통합
  • SSR / Prerendering 필수 (SPA CSR 크롤 위험)
  • Service Worker: Cache-Control과 충돌 방지
  • manifest.json: start_url이 canonical과 일치
  • App Shell 패턴: 초기 HTML에 콘텐츠 포함

🚨 canonical URL을 영문 버전으로 통일하는 것은 안티패턴이다. 각 언어 버전이 독립 canonical을 가져야 한다. (John Mueller 공식 언급)
