# research 에이전트

학술 리서치, 경쟁사 분석, 데이터 분석 설계, 꿈 해석 등 다양한 조사·분석 작업을 수행하는 에이전트 모음.

| 에이전트 | 설명 |
|---------|------|
| [deep-researcher](../../../.claude/agents/research/deep-researcher.md) | 논문/오픈소스/기업사례 3축 딥리서치 오케스트레이터. 리뷰어 품질 평가 포함 |
| [web-searcher](../../../.claude/agents/research/web-searcher.md) | 검색 축(논문/오픈소스/기업사례)별 쿼리를 받아 관련 소스 탐색·구조화 반환 |
| [research-reviewer](../../../.claude/agents/research/research-reviewer.md) | 리서치 보고서 5항목 품질 평가 → PASS 또는 GAPS(부족 항목 + 추가 검색 제안) |
| [academic-researcher](../../../.claude/agents/research/academic-researcher.md) | 인문학·철학 주제 학술 리서치 오케스트레이터. 한국 인문학 인용 형식 구조화 보고서 |
| [literature-review-synthesizer](../../../.claude/agents/research/literature-review-synthesizer.md) | 다수 학술 문헌 → 연구사 표·학파 계보도·핵심 논쟁 매트릭스 생성 |
| [translation-comparison](../../../.claude/agents/research/translation-comparison.md) | 아리스토텔레스 등 1차 텍스트 다역본 행 단위 비교 분석 |
| [defense-question-simulator](../../../.claude/agents/research/defense-question-simulator.md) | 학위논문 → 심사위원 예상 질문 12~15개 + 학술적 모범 답변 |
| [research-proposal-coach](../../../.claude/agents/research/research-proposal-coach.md) | 큰 연구 주제 → RQ 후보 5~8개 생성 + 지도교수 면담용 1쪽 요약 |
| [socratic-interviewer](../../../.claude/agents/research/socratic-interviewer.md) | 모호한 요구사항을 질문으로 이끌어 명확한 명세로 변환 |
| [data-analyst](../../../.claude/agents/research/data-analyst.md) | 이벤트 택소노미·퍼널 분석·A/B 테스트·KPI 대시보드 설계 및 SQL 작성 |
| [competitor-analyst](../../../.claude/agents/research/competitor-analyst.md) | 경쟁사 분석 → 시장 포지셔닝·기능 비교표·차별화 전략 도출 |
| [dream-multi-perspective-synthesizer](../../../.claude/agents/research/dream-multi-perspective-synthesizer.md) | 꿈 한국 전통·프로이트·융·애착 이론 다관점 통합 해몽 리포트 |
| [dream-journal-coach](../../../.claude/agents/research/dream-journal-coach.md) | 누적 꿈 일기 시리즈 분석 + 반복 패턴·감정 분포 + 융 해석 코칭 |
